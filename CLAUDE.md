# Hangzhou Chemnitz 官网维护

这是杭州开姆尼茨国际贸易有限公司的官方网站，域名 hzchemnitz.cn。

## 文件结构
- `index.html` — 英文主页（主要编辑这个）
- `ru.html` — 俄文版
- `ar.html` — 阿拉伯文版
- `fr.html` — 法文版
- `images/` — 图片资源
- `CNAME` — 域名配置（不要动）

## 如何修改
纯静态HTML+CSS，直接编辑HTML文件即可。浏览器打开 index.html 预览效果。

## 如何部署
网站托管在 GitHub Pages。修改完执行：
```bash
git add . && git commit -m "update" && git push
```
推送后GitHub自动部署，几分钟后官网更新。
